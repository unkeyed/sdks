# StringMatch

String matcher. Exactly one of `exact`, `prefix` or `regex` must be set.


## Fields

| Field                                                                                                                   | Type                                                                                                                    | Required                                                                                                                | Description                                                                                                             |
| ----------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| `exact`                                                                                                                 | *Optional[str]*                                                                                                         | :heavy_minus_sign:                                                                                                      | Matches when the input equals this value.                                                                               |
| `prefix`                                                                                                                | *Optional[str]*                                                                                                         | :heavy_minus_sign:                                                                                                      | Matches when the input starts with this value.                                                                          |
| `regex`                                                                                                                 | *Optional[str]*                                                                                                         | :heavy_minus_sign:                                                                                                      | Matches when the input satisfies this RE2 regular expression. Invalid<br/>patterns are rejected when the policy is created. |
| `ignore_case`                                                                                                           | *Optional[bool]*                                                                                                        | :heavy_minus_sign:                                                                                                      | Compare case-insensitively. May accompany any match mode.                                                               |
