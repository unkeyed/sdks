# KeyCreditsRefill

Configuration for automatic credit refill behavior.

## Example Usage

```typescript
import { KeyCreditsRefill } from "@unkey/api/models/components";

let value: KeyCreditsRefill = {
  interval: "daily",
  amount: 1000,
  refillDay: 15,
};
```
