# Deploy

## Overview

Deployment operations

### Available Operations

* [CompleteUpload](#completeupload) - Complete upload

## CompleteUpload

Signals that a deployment build context upload completed.

**Authentication**: Requires a valid root key with appropriate permissions.


### Example Usage

<!-- UsageSnippet language="go" operationID="deploy.completeUpload" method="post" path="/v2/deploy.completeUpload" -->
```go
package main

import(
	"context"
	"os"
	unkey "github.com/unkeyed/sdks/api/go/v2"
	"github.com/unkeyed/sdks/api/go/v2/models/components"
	"log"
)

func main() {
    ctx := context.Background()

    s := unkey.New(
        unkey.WithSecurity(os.Getenv("UNKEY_ROOT_KEY")),
    )

    res, err := s.Deploy.CompleteUpload(ctx, components.V2DeployCompleteUploadRequestBody{
        DeploymentID: "d_abc123",
        BuildContextPath: "proj_123/build_abc.tar.gz",
        DockerfilePath: "Dockerfile",
    })
    if err != nil {
        log.Fatal(err)
    }
    if res.V2DeployCompleteUploadResponseBody != nil {
        // handle response
    }
}
```
