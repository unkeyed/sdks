# Kind

How the value may be read back. Defaults to `writeonly`.


## Example Usage

```go
import (
	"github.com/unkeyed/sdks/api/go/v2/models/components"
)

value := components.KindRecoverable
```


## Values

| Name              | Value             |
| ----------------- | ----------------- |
| `KindRecoverable` | recoverable       |
| `KindWriteonly`   | writeonly         |
